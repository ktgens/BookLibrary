
Проект представляет собой консольное приложение для управления личной библиотекой. Пользователь имеет возможность взаимодействовать с библиотекой через консоль, создавать новые книги, пользователей, отдавать книги пользователям и возвращать их.

# Инструкция по сборке

### Windows

В папке, где был прописал `git clone...`:
```cmake
mkdir build
cd build
cmake ..
cmake --build
.\library.exe
```
